# WordARTs with `figlet`

### WordArt for system names

EPEL has a package `figlet`; it is used to generate the `WordART` of the system names.

```
$ docker run --rm -it almalinux:8
```
Inside the docker contianer, install the `epel-release` package first to enable `EPEL`, then install `figlet` package.

```
dnf install -y epel-release
dnf install -y figlet
```



### Using `figlet`

Now use `figlet` to create word art. Use `-c` to center the text.

NOTE: Creaeted word art may contain(s) \` (`backtick`) needed to be replaced with `single-qote` `'`. This step is needed since backtick is escape commands in shell window.
